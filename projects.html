Personal Projects

Ocact-Chess Spring 2023

For the final project of Cornell's CS 3110 (Functional Programming and Data Structures), I collaborated with three partners to create a terminal based user interface for chess in Objective-Caml. It supports all the standard rules of chess including en-passant, castling, and pawn promotion. Users move a piece [p] by entering the square [p] is on and then the square they want to move [p] to. The game disallows illegal moves and identifies when a game ends in stalemate or checkmate.

I personally implemented the user interface, king and rook mechanics, and an exhaustive test suite. Nearly all development was conducted in Objective Caml (OCaml) and the total lines of code totaled around 1600. In addition to working on the gameplay features, I focused on test-driven development and efficient code in this project.

Critter World Fall 2021

The goal of this project was to simulate a world of critters with the capability to move, eat, reproduce, evolve, and die. It served as the capstone project to CS 2112 (Honors Object Oriented Programming and Data Structures) at Cornell University. The project was split into three major components: first a recursive descent parser for a simple critter language, second a model-controller for critter simulation, and third a graphic user interface (GUI) to visually represent the model. Additionally, a rigorous test suite of parameterized, random, blackbox, and glassbox testing was developed to check the validity of all modules.

I worked with two partners to complete this project, and I was personally responsible for building our abstract syntax tree representation of the "critter language," designing the GUI in JavaFX for our simulation, and maintaining the test suite of all team members' code. Ultimately, the project totaled around 12,000 lines of code in Java and was completed over the span of a month and a half.

Virulence Gene Study Summer 2020

Under Dr. Zuyi Huang of Villanova University, in this project, I studied global trends of virulence genes in antimicrobial-resistant pathogens. In this project, I performed principal component analysis (PCA) and hierarchical clustering in R using data from the National Practictioner Data Bank (NPDB). As the leader of 5 students, I managed project code and spearheaded the write-up of our manuscript. Our research was ultimately published in Processes (impact factor: 2.75), where I received first author recognition.

Through this research, I gained project leadership experience and exposure to advanced statistical methods like PCA and hierarchical clustering. Beyond my code contributions, I was quality checking team members' code and generating key graphics summarizing the findings. Moreover, I became very experienced in technical writing, especially in the context of scientific research.
